Problem Statement

  • Manual attendance systems are time-consuming and prone to fraud
  • Single-factor authentication (ID cards) easily compromised
  • Existing systems fail during network outages
  • High costs limit adoption in resource-constrained institutions
  • Lack of integration between RFID and biometric technologies
  • Poor data synchronization between local and central systems

Research Objectives

  • Develop dual-authentication framework (RFID + Fingerprint)
  • Enable seamless offline and online operation modes
  • Optimize cost-effectiveness using readily available components
  • Automate attendance logging and reporting systems
  • Assess system reliability and usability in real-world scenarios
  • Evaluate broader applications beyond educational institutions

Technical Specifications

Sensor Physics

  • RFID (RC522): Operates at 13.56 MHz using electromagnetic induction
  • Fingerprint Sensor: Capacitive sensing technology for ridge/valley detection

Offline Mode Handling

  • Local storage using SPIFFS/LittleFS file systems
  • SQLite database for structured data management
  • Queue mechanism for pending synchronization
  • Retry logic with exponential backoff for failed uploads
